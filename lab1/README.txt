環境建置:

使用Visual Studio 2022
模式設為 x86
打開sln檔案後至visual studio 屬性頁面點擊 C/C++、一般，其他include目錄中設定以下路徑
至鏈結器(linker)/一般，其他程式庫目錄設定lib路徑
至鏈結器(linker)/輸入，其他相依性設定freeglut.lib



功能:

每個圖形不同顏色

畫點( d )
滑鼠按下時，可以劃出任意圖形，放開時停止

畫線( l )
點擊第一次對應初始座標，點擊第二次對應終點座標。

畫圓( o )
點擊第一次對應圓心，第二次給出圓半徑

多邊形( p )
左鍵點擊對應多邊形的頂點，右鍵點擊完成封閉曲線

清除( c )
清除屏幕上的圖形

undo( u )
回復清除前的狀態

結束程式( q )
關閉視窗
